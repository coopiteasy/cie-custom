Foodhub Customization

Set Mandatory Fields in Product:
- Volume
- Weight
- Vendors
- H.S. code
- Sale Description
- Website Product Category
- Sale Price

When duplicating a product, copy more fields over. The fields are:
- Volume
- Weight
- Vendors
- H.S. code
- Country of origin
